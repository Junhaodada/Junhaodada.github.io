花里胡哨的！

2021年 8月24日

最近整理了一下面向对象的思维导图，还没整理完。

总体感觉，细节的东西还是有很多的！比如下面这些问题，第一次学习的时候都是很少考虑的。

  • 一个对象的创建过程？
  • 为什么构造方法里可以使用this？
  • 为什么static不能使用this？
  • ……

当然难点也有的，目前看是内存分析和垃圾回收机制。

需要重点掌握的东西：对象和类、构造器、三大特征、抽象类和接口、内部类等，决定不能马虎，必须进行代码实现。

2021年 9月28日

今天花费1个多小时的时间，补充完了面向对象三大特性的部分，对封装、继承、多态，有了初步理解。但是想理解透彻的话，还得多项目。
